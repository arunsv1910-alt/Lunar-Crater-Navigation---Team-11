# LUNAR ROVER SIMULATION
## EECE 5550 Mobile Robotics - Final Project
### Team 11: Lunar Crater Navigation and Terrain-Relative Localization

**Simulation Engineer:** Arun Saravana Lakshmi Venugopal

## PROJECT OVERVIEW
This simulation implements the particle filter-based localization framework from ShadowNav (Atha et al., 2024) for lunar rover navigation using crater landmarks.
